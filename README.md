# Mini Katalog Uygulaması

## Kısa Açıklama
Bu çalışma, gerçekleştirilen Flutter geliştirici kampı süresince inşa edilen, temel düzeyde bir e-ticaret vitrin simülasyonudur. Uygulama kapsamında; asenkron veri modellerinin ızgara görünümü (GridView) üzerinde sergilenmesi, pencereler arası güvenli veri iletimi (Route Arguments) ve alışveriş sepeti gibi dinamik süreçlerin StatefulWidget mimarisiyle canlı olarak yönetilmesi gibi temel mobil geliştirme pratikleri sergilenmektedir.

## Kullanılan Sürümler
* Flutter Sürümü: 3.44.2
* Dart Sürümü: 3.12.2

## Çalıştırma Adımları
1. İlgili depoyu yerel diskinize indirmek adına projeyi bilgisayarınıza klonlayın.
2. Projenin kök dizininde bir terminal penceresi açarak gerekli tüm bağımlılıkların indirilmesi için "flutter pub get" komutunu yürütün.
3. Bilgisayarınıza tanımlanmış sanal bir emülatör veya USB hata ayıklama modu aktif fiziksel bir cihaz bağladıktan sonra flutter run komutu yardımıyla uygulamayı derleyip ayağa kaldırın.
