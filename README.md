# Mini Katalog Uygulaması

## Kısa Açıklama
Bu proje, Flutter eğitim kampı kapsamında geliştirilmiş temel bir e-ticaret katalog simülasyonudur. GridView ile ürün listeleme, sayfalar arası navigasyon (Route Arguments ile veri taşıma) ve StatefulWidget kullanılarak basit sepet durumu yönetimi içermektedir.

## Kullanılan Sürümler
* Flutter Sürümü: 3.44.2
* Dart Sürümü: 3.12.2

## Çalıştırma Adımları
1. Projeyi bilgisayarınıza klonlayın.
2. Terminalde proje dizinine giderek `flutter pub get` komutunu çalıştırın ve bağımlılıkları yükleyin.
3. Bir emülatör veya fiziksel cihaz bağladıktan sonra `flutter run` komutu ile uygulamayı başlatın.
